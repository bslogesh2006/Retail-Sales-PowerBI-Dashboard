# 📊 Retail Sales Performance & Business Insights Dashboard

## 📌 Project Overview

This project is a 6-page interactive Power BI dashboard designed to analyze retail sales performance and provide actionable business insights.

The dashboard enables users to explore sales, profit, customer behavior, product performance, shipping metrics, and executive-level KPIs through interactive visualizations and slicers.

---

## 📑 Dashboard Pages

1. Executive Dashboard
2. Sales Analysis Dashboard
3. Profit Analysis Dashboard
4. Customer Analysis Dashboard
5. Product & Shipping Analysis Dashboard
6. Executive Business Insights Dashboard

---

## 📈 Key Features

- Interactive KPI Cards
- Dynamic Slicers
- Sales & Profit Analysis
- Customer Insights
- Product Performance Analysis
- Shipping Analysis
- Executive Business Insights
- Professional Dashboard Design
